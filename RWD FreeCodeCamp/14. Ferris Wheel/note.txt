transform-origin
-> this property is used to set the point around which CSS transformation is applied.

@keyframes
-> To define the flow of a css animation. Within the @keyframes rule, you can create a selectors for a specific points
in the animation sequence, such as 0% or 25%, or use from and to, to define the start and end of the sequence.

animation-name
-> Is used to link  a @keyframes rule to a CSS selector. 
-> The value of this property should match the name of the @keyframes rule.

animation-duration
-> This property is used to set how long the animation should sequence to complete.
-> The time should be specified in either seconds (s) or miliseconds (ms).

animation-iteration-count
-> This property sets how many times your animation should repeat.

animation-timing-function
-> Property used how to set animation should progress over time.
