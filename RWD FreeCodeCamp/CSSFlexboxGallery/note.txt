--> Flexbox

-> flexbox has main and cross axis. The main axis is defined by the flex-direction property which has
   4 possible values:

   1. row (default): horizontal axis with flex items from left to right
   2. row-reverse: horizontal axis with flex items from right to left
   3. colomn: vertical axis with flex items from top to bottom
   4. column-reverse: vertical axis with flex items from bottom to top

   NOTE: The axes and directions with be different depending on the text direction. The values shown are 
   for a left-to-right text direction.


--> Flex-wrap
-> flex-wrap property determines how your flex items behave when the flex container is to small.
-> setting it to wrap will allow the items to wrap to the next row of column.
-> nowrap(default) will prevent your items from wrapping and shrink them if needed.

--> object-fit
-> object-fit property detemine how images should behave.

--> gap
-> gap property also known as gutter between rows and columns. 
-> gap property and its row-gap and column-gap sub properties in this functionality for flex,grid and 
   multi-column layout
