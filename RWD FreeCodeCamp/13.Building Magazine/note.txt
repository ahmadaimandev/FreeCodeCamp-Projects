Lazy loading
-> lazy to tell the browser not to fetch the image resources until it's needed (as
in, when the user scrolls the image view).

-> As an additional benefit, lazy loaded elements will not load until the non-lazy elements are loaded-
this means users with slow internet connection can view the content of your page without having to wait for the images to load.


Referrer
-> The Referrer HTTP header contains information about the address or URL of a page that a user might be visiting from.
-> This information can be used in analytic to track how many users from your page.

Grid Layout
-> CSS grid offer two-dimensional grid based layout. Allowing you to center items
horizontally and vertically while still retaining control to do things like overlap elements.

-> CSS grid similar to flexbox in that it has a special property for both the parent and child elements.

minmax function
-> minmax function allows your columns responsive on any device.
-> minmax takes two arguments
1. First is being the minimum value
2. Second is maximum value
3. This value could be a length, percentage, fr or even keyword like max-content.
